Claude Code
桌面管理面板

可视化管理 Claude Code 的配置、会话、Skills、Plugins、Hooks 和 Memory。
无需手动编辑 JSON / Markdown 文件。

下载最新版本 在 GitHub 查看

仅支持 Windows · 需要 Claude Code CLI

功能模块

所有 Claude Code 配置，一处搞定

所有数据直接读写 ~/.claude/ 下的文件，无额外数据库

仪表盘

多项目卡片网格，显示会话数、活跃状态，30s 自动刷新

Prompt

可视化编辑全局及项目级 CLAUDE.md，CodeMirror + 实时预览

Memory

浏览/新建/编辑/删除 memory/*.md，按 type 过滤，自动同步索引

Sessions

会话历史列表，查看完整消息时间线，含 thinking block 折叠展示

Skills

扫描 ~/.claude/skills/，主从布局查看 SKILL.md，区分本地与符号链接

Plugins

已安装插件管理（启用/禁用）+ 市场浏览，支持关键词搜索

Hooks

可视化编辑 settings.json 中的 hooks，支持五种事件类型完整配置

Settings

全局设置（模型/API Key/权限）+ 项目级 settings.json 编辑器
